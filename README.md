## 📊 笔记分析

使用笔记分析工具可以自动生成笔记统计报告：

```bash
python3 analyze_notes.py
```

这将生成 `笔记分析报告.md` 文件，包含以下内容：
- 📈 总体统计（笔记数量、行数、字数等）
- 📚 主题分类统计
- 📝 详细笔记列表
- 🏆 最详细的笔记排行
- 🔍 内容特征分析
- 💡 学习建议

查看最新的分析报告：[笔记分析报告.md](笔记分析报告.md)
